# Text emphasis
* 굵게
    * `**` | `__`
    * JAVA
    * **JAVA**
    * __JAVA__

* 기울기
    * `*` | `_`
    * PYTHON
    * *PYTHON*
    * _PYTHON_

* 밑줄
    * `<U>` | `</U>`
    * Spring
    * <U>Spring</U>

* 취소선
    * `~~`
    * jQuery
    * ~~jQuery~~
